# # Scriptorium

A pipeline for downloading and archiving ancient and classical texts 
from Project Gutenberg and beyond.

## Architecture
LM Studio (desktop) → Orchestrator (Linux laptop) → Server (storage rig)

## Current Source
- Project Gutenberg

## Planned Sources
- Internet Archive
- Perseus Digital Library

## Stack
- Python


## Goals
- to create a system of agentic download of public texts from ancient
eras, to automate the process of finding certain relevant texts
